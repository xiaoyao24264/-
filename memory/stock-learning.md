## 一、股票基础概念

### 什么是股票？
- 股票是公司发行的所有权凭证
- 持有股票 = 持有公司的一部分（股东）
- 股票可买卖，价格随市场波动

### 核心术语
| 术语 | 解释 |
|------|------|
| A股 | 中国境内上市的以人民币交易的股票 |
| B股 | 以人民币标明面值，外币认购和买卖 |
| 主板 | 大型成熟企业（600/000开头） |
| 创业板 | 创新型企业（300开头） |
| 科创板 | 科技创新企业（688开头） |

## 二、K线图（蜡烛图）

### 组成要素
- **开盘价**：交易日第一笔成交价格
- **收盘价**：交易日最后一笔成交价格
- **最高价**：当日最高成交价
- **最低价**：当日最低成交价

### K线类型
- **阳线**（红/上涨）：收盘价 > 开盘价
- **阴线**（绿/下跌）：收盘价 < 开盘价
- **上影线**：最高价到收盘价/开盘价
- **下影线**：最低价到开盘价/收盘价

## 三、均线系统

### 常用均线
- MA5：5日均线（周线）
- MA10：10日均线（半月线）
- MA20：20日均线（月线）
- MA60：60日均线（季线）
- MA120：120日均线（半年线）
- MA250：250日均线（年线）

### 均线用法
- **金叉**：短期均线上穿长期均线 → 买入信号
- **死叉**：短期均线下穿长期均线 → 卖出信号
- **支撑**：价格下跌到均线附近反弹
- **压力**：价格上涨到均线附近回落

## 四、成交量

### 量价关系
- **量升价涨**：健康上涨，后市可期
- **量升价跌**：可能下跌趋势中继
- **缩量上涨**：惜售，可能继续涨
- **缩量下跌**：可能见底

### 地量与天量
- **地量**：成交量极度萎缩，变盘信号
- **天量**：成交量异常放大，注意风险

## 五、常用技术指标

### MACD
- DIFF线与DEA线的交叉
- 金叉买入，死叉卖出
- 零轴上方多头，下方空头

### RSI（相对强弱指标）
- 0-100波动
- >80 超买，可能下跌
- <20 超卖，可能上涨

### BOLL（布林带）
- 上轨、中轨、下轨
- 价格突破上轨可能回调
- 价格跌破下轨可能反弹

## 六、交易规则

### 交易时间
- 集合竞价：9:15-9:25
- 连续竞价：9:30-11:30, 13:00-15:00

### 涨跌幅
- 主板/创业板：10%
- 科创板：20%
- ST股：5%

### 交易单位
- 100股 = 1手
- 最低买入100股
