# 股票进阶知识

## 八、趋势分析

### 趋势类型
- **上升趋势**：高点越来越高，低点也越来越高
- **下降趋势**：高点越来越低，低点也越来越低
- **横盘整理**：价格在区间内波动

### 趋势线
- 连接两个以上低点 → 上升趋势线（支撑）
- 连接两个以上高点 → 下降趋势线（压力）

## 九、形态分析

### 常见反转形态
- **头肩顶/底**：趋势反转信号
- **双顶/双底**：M头/W底
- **圆顶/圆底**：缓慢反转

### 常见持续形态
- **旗形**：短暂整理后继续
- **三角形**：收敛后突破
- **矩形**：区间震荡

## 十、基本面分析

### 重要指标
| 指标 | 含义 | 参考值 |
|------|------|--------|
| 市盈率(PE) | 价格/每股收益 | 20-30合理 |
| 市净率(PB) | 价格/每股净资产 | 1-5合理 |
| ROE | 净资产收益率 | >15%优秀 |
| 毛利率 | 盈利能力 | 越高越好 |

### 财报时间
- 一季报：4月
- 中报：7-8月
- 三季报：10月
- 年报：次年1-4月

## 十一、仓位管理

### 仓位原则
- **不满仓**：预留应对突发情况
- **分批建仓**：不要一次性买入
- **金字塔式**：越跌越买，越涨越卖

### 仓位控制
- 新手建议：不超过30%仓位
- 经验者：不超过50-70%仓位
- 永远留有余地

## 十二、心态管理

### 常见心态问题
- 追涨杀跌
- 频繁交易
- 亏损后死扛
- 盈利后拿不住

### 建议
- 制定交易计划
- 严格执行纪律
- 接受亏损是常态
- 保住本金最重要
